## 🧪 A/B test strategie

**Week 1 (€15-25/dag totaal, ~€3-5/ad):**
- Alle 5 varianten parallel laten lopen
- Identieke audience targeting
- Doel: zien welke 1-2 ads het meeste CTR + form-fill geven
- KPI's:
  - CTR > 1,5% = goede ad
  - Cost per landing page view < €1,50
  - Cost per form-fill (demo-aanvraag) < €30 = sterk start

**Week 2:**
- Stop varianten onder 0,8% CTR
- Verdubbel budget op winnaars (€10/dag per overlevende ad)
- Voeg nieuwe creatives toe op de winnende hook (variant met andere foto, A/B test images)
- Start NL-locatie

**Week 3-4:**
- Lookalike-audience van form-fillers
- Retargeting: bezoekers die geen demo aanvroegen, met case study aanbod
- Eerste evaluatie totale CAC (cost per nieuwe klant) — moet onder €500 blijven voor unit economics

## 💡 Quentin framework toegepast

Voor elke variant: **vind het lek → installeer agent → koppel → 24/7 → meet**

- V1 = lek: gemiste calls op werf
- V2 = meet: 87% / 90% / 27pp / €0 cijfers
- V3 = koppel + 24/7: monitor vs. DIY-onderhoud
- V4 = installeer: voicebot in jouw stack
- V5 = meet: 4u/dag terug, zaakvoerder-perspectief

Geen US-hype, geen "AI EMPLOYEE BUILD 72%" — gewoon outcome-taal in Vlaamse B2B-toon.
